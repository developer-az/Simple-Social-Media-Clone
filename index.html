🧠 Scientific UX Principles Applied

1. Visual Hierarchy

Typography scale (1.25 ratio) creates clear information hierarchy, guiding user attention naturally.

2. Fitts's Law

Buttons sized 44px+ for easy targeting on all devices. Closer targets for frequent actions.

3. Color Psychology

Blue promotes trust, green for positive actions, red for warnings. 4.5:1 contrast ratio for accessibility.

4. Gestalt Principles

Related elements grouped using proximity and similarity for intuitive understanding.

5. Cognitive Load Theory

Maximum 7±2 elements per group. Progressive disclosure for complex actions.
